# Changelog

## [0.1.0] - 2026-06-20

### Added
- Initial Get Node connector: `node://get/installer/query/script` (returns the
  get.ifuri.com bootstrap command without a network call) and
  `node://host/runtime/query/health` (local urirun runtime health).
- CLI (`urirun-get-node`), connector manifest, pytest suite, smoke target,
  `urirun.bindings` entry point and GitHub Actions CI.
